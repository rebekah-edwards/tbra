# tbr(a) — Sample Book Page (wireframe v0)

Goal: one page that proves the value prop: *"Tell me what's in this book."*

---

## Header
- Title
- Author(s) (tap → author page later)
- Publication year
- Genre chips
- **ISBN** (small, collapsible: “Details ▾”)

## Description (no spoilers)
- Auto-populated description
- “Show more” expand

## Quick Actions
- Add to TBR
- Add to Owned
- State: Currently Reading / Completed / Paused / DNF

## Content Profile (the product)
For each category: show intensity bar (0–4) + short notes if ≥2.

### Categories (v0)
- LGBTQIA+ representation
- Religious content
- Witchcraft / occult
- Sexual content
- Violence & gore
- Political & ideological content
- Profanity / language
- Substance use
- Self-harm / suicide
- Sexual assault / coercion
- Child harm

### Evidence badge (small)
- AI Inferred / Cited / Human Verified

### Optional: “Why we think this” ▾
- Hidden by default
- Shows citations (links/quotes/locators) *for disputed categories first* (suggested: Ideological content, LGBTQIA+ rep)

## Links
- Amazon
- Publisher
- Pre-save (if upcoming)

## Reviews (v0: minimal)
- Either: external links only, or a small “review snippets” section later.

## Report a correction
Button (always visible) → modal:
- Which category? (optional)
- Proposed intensity (0–4)
- Proposed notes
- Optional citation link
- Freeform explanation

---

## MVP guardrails
- No social feed.
- No comments.
- No recommendations until we have enough profiles + a reading profile definition.
- Optimize for speed of adding profiles + trust in profiles.
